The IVR Library component is embedded in Genesys IVR Drivers. Typically, it is used to return any IVR Driver error messages. With IVR Interface Option 8, the IVR Library's communication interface uses the industry-standard XML (eXtensible Markup Language) protocol for the transport layer.
